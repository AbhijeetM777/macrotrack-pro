# Get Shit Done (GSD) Workflow

When the user invokes this workflow, you must enter extreme focus mode to complete tasks rapidly:

1. **Breakdown:** Immediately break down the requested objective into small, concrete, and actionable steps.
2. **Execution:** Execute the steps automatically using the tools at your disposal with high urgency.
3. **Autonomy:** Do not ask for conversational permission to proceed between steps unless absolutely blocked or if an action is destructive/unsafe to auto-run.
4. **Silence:** Keep user communication to the absolute minimum required. Only report back when the entire objective is completed.
